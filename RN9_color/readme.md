Aplikasi grafis 3D berbasis WebGL. Karakter diberikan warna yg berbeda (pink , ungu, dan biru) dan dirender menggunakan phong lighting model untuk simulasi pencahayaan realistis (komponen ambient, diffuse, dan specular). Dapat menangani transformasi geometri, kalkulasi vektor normal, dan proyeksi perspektif, serta terdapat fitur interaktif agar pengguna dpt melakukan rotasi objek sesuai keinginannya
